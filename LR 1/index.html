Розробка простої веб-сторінки з використанням HTML5

World Wide Web (WWW, web, веб, Всесвітня павутина) - це розділена інформаційна
система надає доступ до гіпертекстових документів за протоколом HTTP (HyperText
Transfer Protocol). WWW- це мережева технологія, побудована на клієнт-серверній архітектурі, яка використовує інфраструктуру використовує інфраструктуру Інтернет для взаємодії між сервером і клієнтом (Мал. 1)

Общая структура

        <html>
          <head>
            <title>section</title>
          </head>
          <body>
            Some text
          </body>
        </html>
